== Description ==
**WP and PHP MEMORY MONITORING AND ENHANCING MEMORY USAGE**
★★★★★<br>
> Plugin para monitorar o uso de memória projetado para detectar e registrar erros de alto uso de memória em seu site WordPress. Com sua integração perfeita na página de saúde do site, você pode acessar facilmente o uso de memória e os limites de memória e ser alertado quando surgirem problemas. 
> Versão Premium: Diga adeus à codificação complexa e ao risco de alterações acidentais de arquivos. O WP Memory Check permite que você corrija facilmente o limite de memória do PHP e o limite de memória do WordPress sem a necessidade de modificações de codificação intrincadas ou o medo de alterações de arquivos não intencionais. 
> Mas isso não é tudo! Nosso plugin também exibe a RAM total do seu servidor, mostrando a memória física instalada. 
> Tome medidas e instale o plugin WPmemory hoje para se libertar das restrições de memória. Com o WPmemory, você pode se concentrar no que realmente importa: criar conteúdo notável, envolver seu público e desbloquear seu potencial criativo. 
> Pronto para vários idiomas: arquivos em inglês, italiano, português e espanhol incluídos. 
. 
> Português: Plugin para verificar se há alto uso de memória. Todo o interface pode ser visto em Português. 
. 
> Italiano: Plugin per verificare l'utilizzo elevato della memoria. L'intera interfaccia può essere visualizzata in Italiano. 
. 
> Español: Plugin para verificar el alto uso de memoria. Toda la interfaz se puede ver en Español. 
Requisitos de memória do WordPress: nunca mais receba o erro "Erro fatal: tamanho de memória permitido de xxx bytes esgotado". Muito fácil de instalar e gerenciar. Basta ir para a página do seu plugin para ativar/desativar. Vá para a página de saúde do site (Painel => Ferramentas => Saúde do site - Status) e procure por Uso de memória. Vá para o Painel => Ferramentas => WP Memory para o painel e a página de ajuda com dicas e sugestões. Para ajudá-lo com a depuração, procure também a página de informações (Painel => Ferramentas => Saúde do site - Informações). Procure por Uso de memória. Mostra os erros do WordPress (erros PHP error_log, log de erros ou relatório de erros). Erros fatais, avisos, alertas e erros de análise ou sintaxe para ajudá-lo a descobrir e corrigir problemas. Você precisa do WordPress 5.2 ou superior para instalar este plugin.

== Instalação == 
1) Instale via wordpress.org 
2) Ative o plugin pelo menu 'Plugins' no WordPress ou extraia o arquivo zip e coloque o conteúdo no diretório wp-content/plugins/ da sua instalação do WordPress e, em seguida, ative o plugin na página Plugins.
